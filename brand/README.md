# Brand assets (placeholder)

`icon.png` (256×256) and `icon@2x.png` (512×512) are **placeholder** solid-color
icons so the integration has a brand asset.

For the icon to render in Home Assistant / HACS, brand images must be submitted to
the official [`home-assistant/brands`](https://github.com/home-assistant/brands)
repository under `custom_integrations/minecraft_broker/`. Replace these placeholders
with a real logo before submitting. Installation via a HACS custom repository works
without this step.
